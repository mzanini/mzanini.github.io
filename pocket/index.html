This page shows a simple javascript application that connects to Pocket, retrieves the list of read items and puts them in a list. Items are sorted with the newest item first. All you need to do is to create a new application on the Pocket developer portal with Retrieve permission. Then input the consumer key in the box and click Show read items. Apologies if this step is not automatic but I am trying to find out how to make everything work with the limitations of github pages, where this website is currently hosted.

The consumer key is just saved in your browser's session storage
